get /products/

Get a list of products

  • Request
  • Response

Query Parameters

  • name: (string)

    Product name

    Example:

    Produto A
  • sku: (string)

    Product sku

    Example:

    fd5fac04-f4af-4c02-b7ec-0d0d0c50af78
  • fromPrice: (number)

    Minimum price

    Example:

    5
  • toPrice: (number)

    Maximum price

    Example:

    10.99
  • fromStock: (integer)

    Minimum stock

    Example:

    5
  • toStock: (integer)

    Maximum stock

    Example:

    10
  • fromDate: (string)

    Minimum creation date (format: yyyy-mm-dd)

    Example:

    2021-02-01
  • toDate: (string)

    Maximum creation date (format: yyyy-mm-dd)

    Example:

    2021-02-15
  • page: (integer - default: 1)

    Page number

    Example:

    2
  • sort: (string - default: id:asc)

    Property to sort result (format: property_name:[asc/desc])

    Example:

    price:desc
